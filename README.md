# Summary
This is the repository for the Community Website of the "Neubaugebiet Claudiussee".

# Purpose
It should serve informatin for the Claudiussee Neubaugebiet. 
And it should show different Pictures of the Development of the Claudiussee Neubaugebiet.

Shows Images of the development of the Claudiussee Neubaugebiet.
Shows relevant Documents for the Claudiussee Neubaugebiet, like:
- Bebauungsplan
- Entwässerungsplan
- Begrünungsplan
- ...
